## 개인 포트폴리오 웹사이트입니다.

🔗 **Live Demo:** [https://dongim-02.github.io](https://dongim-02.github.io)

<br/>

## 🎨 Design Concept
- **Clean & Professional Light Theme:** 이력서 및 노션(Notion) 문서와 같은 전문적인 느낌을 주기 위해 따뜻한 아이보리 배경과 차분한 골드/네이비 포인트 컬러를 적용했습니다.
- **Responsive Web Design:** 데스크탑, 태블릿, 모바일 등 다양한 디바이스 환경에 최적화된 반응형 레이아웃을 구현했습니다.

<br/>

## 🛠️ Tech Stack
- **Languages:** HTML5, CSS3, JavaScript
- **Deployment:** GitHub Pages (Automated Build & Deploy)
- **Features:** CSS Grid/Flexbox Layout, Intersection Observer API (Scroll Animations)

<br/>

## 🎯 Key Sections
- **Hero & About Me:** 직관적인 타이포그래피와 애니메이션, 그리고 응용 소프트웨어 개발자로서의 비전과 연락처/링크드인 네트워크 정보 제공
- **Tech Stack:** 프론트엔드(UI/UX) 및 AI, IoT, 시스템 보안 역량의 시각적 분류
- **Projects & Activities:** 기획부터 하드웨어 제어, AI 파이프라인 구축까지 아우르는 4대 핵심 프로젝트 타임라인
  - *AutoFactory ROS2* (ROS2, FreeCAD, Step Motor)
  - *Drive: RC카 원격 제어* (WebSocket, ToF, ESP32/STM32)
  - *Virtual Try-On System* (Python, AI Pipeline, FastAPI/React)
  - *IoT 보안 / Co-EDR 기반 펌웨어 퍼징 시스템* (KCI 등재)
- **SWOT & Vision:** 개발자로서의 객관적인 강점/약점 분석 및 실질적 문제 해결을 향한 향후 비전 제시

<br/>

## 📂 Repository Structure
- `index.html`: 웹사이트의 메인 구조 및 콘텐츠, CSS 스타일링, 스크롤 반응형 JS 로직이 통합된 단일 페이지 애플리케이션(SPA) 코드
