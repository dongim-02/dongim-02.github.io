Robotics · ROS2
AutoFactory ROS2 — 자동화 팩토리 시스템

대한상공회의소 서울기술교육센터에서 진행한 프로젝트로, ROS2 기반 자동화 팩토리 시스템을 구현하였습니다. FreeCAD를 활용하여 알약 디스펜서의 구조를 3D 모델링하고, 스텝 모터 정밀 제어 모듈을 구현하여 공정 자동화 로직을 완성했습니다.

ROS2 Python FreeCAD Step Motor
Robotics · WebSocket
Drive — WebSocket 기반 RC카 원격 제어 시스템

대한상공회의소 서울기술교육센터에서 진행한 프로젝트로, 웹 대시보드 및 음성 인식을 통해 RC카를 실시간으로 조종하는 저지연 제어 시스템을 구축했습니다. 통신(ESP32)과 제어(STM32)를 분리한 이중화 MCU 아키텍처를 채택하였으며, 수신 데이터 노이즈로 인한 패킷 유실 문제를 '스트리밍 파싱(Streaming Parsing)' 방식을 도입해 완벽히 해결했습니다. 본인은 ToF 센서 연동 및 라즈베리파이 카메라 모듈 테스트, 웹 실시간 연동 구성을 핵심적으로 수행했습니다.

WebSocket C/C++ ESP32 / STM32 UART 통신 ToF 센서 Raspberry Pi
AI · Data Pipeline
Virtual Try-On System — 가상 피팅 데이터 파이프라인

대한상공회의소 서울기술교육센터에서 진행한 프로젝트로, 사용자 이미지 기반 가상 의류 피팅 시스템 구현 프로젝트입니다. 본인은 의류 이미지 자동 크롤링 및 데이터 전처리 파이프라인 구축을 담당했습니다. 이미지 리사이즈부터 OpenPose(골격 인식), SCHP(인체 파싱), 마스크 생성으로 이어지는 10단계 프로세스를 세분화하여 설계하고, StableVITON 모델 학습에 적합한 데이터셋으로 자동 정제하는 백엔드 파이프라인을 성공적으로 통합했습니다.

Python FastAPI / React PyTorch OpenCV StableVITON
Industrial Automation · PLC
Mini-Project — PLC 기반 자동화 설비 제어 시스템

대한상공회의소 서울기술교육센터에서 진행한 프로젝트로, Mitsubishi PLC(GX-Works2)를 중심으로 서보 모터·센서·알람 지시기·HMI(GOT 터치스크린)를 통합해 자동/수동 겸용 자동화 설비 제어 시스템을 구현했습니다. 본인은 PLC 래더 코드 작성 및 HMI(GT-Designer3) 화면 설계를 중심으로, 전체 시스템은 공동으로 진행했습니다. BSFL, MOV/DMOV 등 래더 로직으로 공정 순차 제어와 서보 위치 티칭 흐름을 구현하고, 실제 장비 시연으로 동작을 검증했습니다.

Mitsubishi GX-Works2 PLC 래더 로직 GT-Designer3 Servo 제어
IoT Security · KCI
Co-EDR 기반 펌웨어 퍼징 시스템

KISIA 융합보안크루 1기 조장으로서 다수 IoT 장치 협력 기반의 EDR 기법을 활용하여 펌웨어 퍼징 효율성 및 패치 검증 신뢰성 향상을 연구했습니다. 단일 장치가 놓칠 수 있는 취약점을 교차 수집하고, 오류 이벤트를 유발한 인풋 기반으로 퍼징을 수행하여 기존 대비 퍼징 효율성을 최대 41.8% 증가시키고 탐지 시간을 99.9% 단축했습니다. 이 성과로 KISTI 원장상을 수상하였으며, 연구 논문이 KCI에 등재되었습니다.

Firmware Fuzzing EDR 기법 IoT Security KCI 논문

IoT 보안 — 침입 감지 이메일 알림 시스템

백석대학교 'IoT보안프로그래밍' 교과목에서 진행한 실시간 침입 감지 및 자동 이메일 알림 전송 파이프라인 구축 프로젝트입니다. 본인은 아이디어 제안, 초기(변경 전) 시스템 구현, 카메라 촬영 파트 구현을 맡았습니다. Raspberry Pi와 PIR 센서로 침입을 감지하고 카메라로 현장을 촬영한 뒤, 이미지 파일을 MIMEBase 객체로 읽어와 Base64 방식으로 인코딩하여 SMTP 서버를 통해 즉각적인 경고 이메일을 전송하는 시스템을 완성했습니다.

Python Raspberry Pi PIR Sensor SMTP MIMEBase / Base64
