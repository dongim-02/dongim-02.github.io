Digital Forensics

초기 디스크 이미지와 다양한 작업 수행 후의 디스크를 비교 분석하였습니다. Autopsy 도구를 활용하여 웹 브라우징 기록(YouTube, Naver 접속 이력), 쿠키, 히스토리, 파일 생성 시간 등의 디지털 증거가 어떻게 기록·보존되는지 분석 시나리오를 수행했습니다.

Autopsy Disk Imaging 포렌식 분석 Windows 포렌식
Research Paper · KCI
고효율 고신뢰 펌웨어의 최적화된 퍼징을 위한 이벤트 데이터 레코딩 기법

EDR(Event Data Recording) 기법을 활용해 펌웨어 퍼징 과정에서 크래시를 유발했던 입력 값을 기록하고, 패치 검증 시 재활용하여 퍼징 효율을 높이는 방법론을 연구했습니다. XiaoGangZhu, Stanislav Abaimov, Jisoo Jang 등의 선행연구를 분석하고 이를 발전시킨 KCI 등재 논문입니다.
